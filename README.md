# cybersecurity-journey
# Do Estágio ao Blue Team Sênior

Repositório criado para documentar minha jornada de aprendizado em cibersegurança :)

Meu plano de desenvolvimento profissional ficou dividido em três fases:

---

## Estágio / Blue Team Júnior
  Período: 0 a 6 meses

Foco:  
Aprender fundamentos técnicos essenciais, iniciar práticas em SOC e documentar tudo.

Metas:
- [x] Criar conta no GitHub e organizar repositório
- [ ] Estudar fundamentos de redes (IP, DNS, OSI)
- [ ] Comandos básicos em Linux (ls, cat, cd, chmod)
- [ ] Estudar logs: event viewer (Windows) e journalctl (Linux)
- [ ] Usar SIEM (Wazuh ou Splunk) para ver alertas
- [ ] Fazer labs no TryHackMe:  
  - [ ] Introduction to Cybersecurity  
  - [ ] SOC Level 1
- [ ] Criar playbook simples de resposta a incidentes
- [ ] Publicar primeiros relatórios no GitHub

Status: Em andamento

---

## Nível Intermediário (SOC N2)
  Período: 6 a 18 meses

Foco:  
Investigar ameaças com mais autonomia, aprender hunting, frameworks e automatizar tarefas.

Metas:
- [ ] Estudar MITRE ATT&CK e aplicar em labs
- [ ] Automatizar tarefas com Bash ou Python básico
- [ ] Criar alertas personalizados em SIEM
- [ ] Simular ataque com metasploit ou logs de brute force
- [ ] Usar EDR (CrowdStrike, Defender) em simulações
- [ ] Fazer playbook de resposta completa

Status: Planejado para 2025

---

## Nível Sênior
  Meta: 18 a 36 meses

Foco:  
Conduzir incidentes ponta a ponta, liderar processos, aplicar GRC e participar de auditorias.

Metas:
- [ ] Criar e revisar playbooks avançados
- [ ] Trabalhar com Nessus, OpenVAS ou Qualys
- [ ] Implementar políticas de GRC (ISO 27001, NIST, CIS Controls)
- [ ] Criar um laboratório próprio com SIEM, endpoints e ataques simulados
- [ ] Documentar resposta a incidente complexa (com relatório final)

Status: Objetivo para 2026

---

Roadmap será atualizado mensalmente conforme avanço nos estudos e projetos.
